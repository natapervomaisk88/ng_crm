Остання версія фреймворку

Підказка до попереднього тексту зроблена через One Way Binding (Малюнок зверху додано так само)

One-way binding в Angular - це механізм, за допомогою якого дані передаються з компонента до шаблону (виду) і підтримуються тільки в одному напрямку, з компонента до шаблону. Це означає, що зміни, внесені до даних в компоненті, автоматично відображаються в шаблоні, але не навпаки.

Pipes

Pipes -вказують на функціональні елементи, які використовуються для обробки та відображення даних у шаблонах Angular. Pipes надають можливість виконувати різні операції перетворення даних без необхідності змінювати їх вихідний вигляд.

  • Pipe uppercase {{ pipes_test[0] | uppercase }}
  • Pipe Date {{ pipes_test[1] | date : "yyyy-MM-dd" }}
  • DatePipe {{ pipes_test[1] | date : "HH:mm" }}
  • CurrencyPipe {{ pipes_test[2] | currency : "USD" : "symbol" : "1.2-2" }}
  • NumberPipe {{ pipes_test[3] | number : "1.2-2" }}
  • PercentPipe {{ pipes_test[4] | percent }}
  • SlicePipe {{ "Angular/React" | slice : 0 : 7 }}

Створення власних Pipes

1) Генерація: ng generate pipe /pipes/title

2) Підключення в imports

{{ "Custom Pipe" | title }}
